Bieber - React Personal Portfolio Template Documentation

Introduction

  • Bieber - React Personal Portfolio Template
  • Created: August 11, 2022
  • Version: 1.0.0
  • By: NuclearThemes

Bieber - is a creative personal portfolio React template build with NextJS and Tailwindcss. This template is designed for building beautiful clean personal portfolio website. It uses all updated packages. It has a great customizable option. By the power of tailwindcss. It is very easy to change any layout and design. If you like creative design this can be a best choice for you.

Setup Development Environment

In order to kickstart the development of your web app with Bieber, the very first thing you need to do is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site. (We prefer latest LTS version.) http://nodejs.org/

Preparation

You'll need to install Node.js >= v15+ (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. powershell or terminal and navigate to the project (bieber) root. This is powershell in windows and terminal on mac/linux.

Let's see how the setup works:

Step-1: Use terminal and navigate to the project root.

Step-2: Rename .env.template to .env and set the environment variables.

Step-3: Then run : npm install or, yarn install
(If you fall any problem with it, try npm install --save --legacy-peer-deps)

Step-4: Then run : npm run dev or, yarn dev

Step-5: Now in the browser go http://localhost:3000

Deploy

For static generation:

  • Make sure your environment variables set properly in .env file. The NEXT_PUBLIC_SITE_URL should be the url where you want to live your site and the NEXT_PUBLIC_API_URL should be your api link. Example:
    NEXT_PUBLIC_SITE_URL=https://nuclearthemes.com
    NEXT_PUBLIC_API_URL=https://nuclearthemes.com/api
  • Run npm run export or, yarn export for build the files.
  • You will find an /out directory in the root directory.
  • Upload the files of the /out directory to your server.

For nodejs server:

  • Follow this link https://nextjs.org/docs/deployment
